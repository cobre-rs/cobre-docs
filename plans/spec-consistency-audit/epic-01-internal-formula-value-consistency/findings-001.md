### Priority Recommendations

1. **HIGH -- Fix SC-1**: Replace $\beta_h^{storage} = \pi^{wb}_{h,1}$ with $\pi^v_h = \pi^{wb}_{h,1}$ in block-formulations.md line 83. This is a pre-epic-06 remnant using $\beta$ instead of the canonical $\pi$ for cut coefficients.

2. **HIGH -- Fix SC-3**: Replace $\pi_h^{balance}$ with $\pi^{wb}_h$ and $\pi_{\hat{v}_h}$ with $\pi^v_h$ in hydro-production-models.md line 281, to match cut-management.md and notation-conventions section 5.

3. **MEDIUM -- Fix MD-10**: Change $c^{th}_{t,s}$ to $c^{th}_{j,s}$ in notation-conventions section 3.2 line 102, since $t$ is the stage index and all math specs use $j$ for the thermal plant index.

4. **MEDIUM -- Add MD-1, MD-2, MD-3, MD-4**: Add $\mathcal{R}$ (non-controllable sources), $\mathcal{C}$ (unified contracts), $\mathcal{H}^{fpha}$, $\mathcal{H}^{const}$ to notation-conventions section 2 index sets table.

5. **LOW -- Add MD-5 through MD-9**: Add the penalty cost parameters ($c^{ctr}_c$, $c^{fpha}_h$, $c^{curt}_r$, Category 2 penalties, $\gamma_j$) to notation-conventions section 3, or add a cross-reference note in section 3 pointing to lp-formulation.md section 1 as the authoritative source for penalty parameters.

6. **LOW -- Document SV-6**: Add a note in notation-conventions section 2 establishing the bus-subscript convention for restricting index sets (e.g., $\mathcal{H}_b \subseteq \mathcal{H}$ = hydros connected to bus $b$).

7. **LOW -- Address SC-2 $\eta$ overloading**: Either rename the PAR innovation noise symbol from $\eta_t$ to something else (e.g., $\xi_t$ or $\varepsilon_t$, following the SDDP.jl convention), or document the dual usage in notation-conventions with a disambiguation note. Also consider whether $\gamma$ overloading (FPHA vs. pumping) warrants a rename for the pumping parameter.
